# 🗺 Roadmap TLDW

## ✅ v1.0 — MVP

- [x] Транскрибация через faster-whisper (small, int8, CPU)
- [x] Извлечение аудио через FFmpeg
- [x] Структурированное саммари через DeepSeek
- [x] Drag & drop загрузка до 2 ГБ, видео + аудио форматы
- [x] Прогресс-бар, статусы, история задач с удалением
- [x] HTTP basic auth через nginx, оценка времени по RTF
- [x] Авто-сброс зависших задач, systemd автозапуск, CLI `tldw`
