# 🗺 Roadmap TLDW

## ✅ v1.0 — MVP (текущее состояние)

- [x] Транскрибация через faster-whisper (small, int8, CPU)
- [x] Извлечение аудио через FFmpeg
- [x] Структурированное саммари через DeepSeek
- [x] Drag & drop загрузка до 2 ГБ
- [x] Видео + аудио форматы
- [x] Прогресс-бар загрузки, статусы обработки
- [x] История задач с удалением
- [x] HTTP basic auth через nginx
- [x] Оценка времени обработки с таймером
- [x] Авто-сброс зависших задач при перезапуске
- [x] systemd автозапуск
- [x] CLI-утилита `tldw`

## 🔴 v1.1 — Безопасность и стабильность

- [ ] **HTTPS через Let's Encrypt** — пароль сейчас в base64 по HTTP. Требует домен.
- [ ] **Автоматический бэкап БД** — cron-задача в S3 или внешний диск
- [ ] **Health endpoint `/health`** — для внешнего мониторинга (Uptime Kuma)
- [ ] **Лимит на размер файла в FastAPI** — сейчас только на уровне nginx
- [ ] **Watchdog для воркера** — сброс задач застрявших >2 часов

## 🟡 v1.2 — Удобство

- [ ] **Экспорт в .txt** — plain text транскрипт + саммари
- [ ] **Экспорт в .docx** — форматированный Word документ
- [ ] **Экспорт в .srt** — субтитры из таймкодов Whisper
- [ ] **Экспорт в .md** — markdown с разделами
- [ ] **Загрузка по URL** — YouTube/VK через `yt-dlp`
- [ ] **Поиск по истории** — SQLite FTS5
- [ ] **Аудиоплеер** с кликабельными таймкодами

## 🟢 v2.0 — Масштабирование

- [ ] **Celery + Redis** — параллельная обработка нескольких задач
- [ ] **Диаризация (pyannote)** — разделение по спикерам
- [ ] **GPU поддержка** — переход на `device="cuda"` для x10 ускорения
- [ ] **Multi-user** — отдельные аккаунты с историей у каждого
- [ ] **REST API** — для интеграций
- [ ] **Telegram-бот** — альтернативный интерфейс через aiogram

## 💡 Идеи на потом

- Распознавание языка автоматически, мультиязычные саммари
- Кастомизация промпта саммари через UI
- Сравнение нескольких видео в одно общее саммари
- Webhook-уведомления когда задача готова
- Sharing — ссылка на саммари для коллег без логина
